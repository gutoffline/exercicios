Imagens

Imagens são fundamentais para bons sites. Além de trazer um elemento estético elas comunicão de forma mais eficiente (normalmente né). Afinal já dizia o ditado:
Uma imagem vale mais que mil palavras.

A seguir, veremos os tipos de imagens mais comuns.

  • PNG - a grande vantagem desse formato de imagem é permitir o fundo transparente
  • JPG - a grande vantagem desse formato é sua compactação, então é possível ter uma imagem com grande qualidade e um tamanho físico baixo se comparado ao PNG por exemplo.
  • GIF - utilizado normalmente com imagens animadas ou imagens mais simples como logotipos ou desenhos
